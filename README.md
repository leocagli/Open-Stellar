# Open-Stellar
Open Stellar - AI Moltbot on Cloudflare Workers with Groq

## Setup

To configure your local repository to push to Open-Stellar, see the [Setup Guide](SETUP.md).

### Quick Start

```bash
cd /workspaces/moltworker
git remote add stellar https://github.com/leocagli/Open-Stellar.git
git push -u stellar main
```

For detailed instructions and troubleshooting, refer to [SETUP.md](SETUP.md).
